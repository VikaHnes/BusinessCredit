Про сервіс

BusinessCredit — це навчальний вебсайт, створений для розрахунку основних параметрів бізнес-кредиту. Користувач може ввести суму кредиту, термін кредитування та річну відсоткову ставку, після чого сайт автоматично обчислює щомісячний платіж, загальну суму виплат та суму переплати.

Сайт реалізований за допомогою HTML, CSS та JavaScript без використання серверних технологій, баз даних або API. Усі обчислення відбуваються на стороні клієнта.

Основна мета проєкту — продемонструвати створення багатосторінкового сайту з інтерактивною функціональністю та зручною навігацією.
